# H2OStore

##Onde a Performance Flui.

O projeto H2O Store foi desenvolvido com html, css e js, para a conclusão da disciplina de 'autoria web' no IFRN, todas as paginas estão funcionais e responsivas, contando com uma pagina de contato com um formulario que chega as respostas para o whatsapp de um membro do grupo por meio de JS.

Intruções de como rodar: 'Clone o repositorio com 'git clone https://github.com/r4viel/H2OStore.git', após isso rode a pagina 'index.html', com seu VScode.'
